IP Header Fields

  • Version: IPv4 or IPv6
  • IHL: Header Length
  • DSCP/ECN: Traffic Class
  • Total Length: IP Packet Size
  • Identification & Fragment Offset: Used for packet fragmentation
  • TTL: Time to Live (prevents infinite loops)
  • Protocol: Layer 4 protocol (e.g. TCP, UDP)
  • Header Checksum: Error detection
  • Source & Destination: IP addresses
  • Options: Optional routing info
